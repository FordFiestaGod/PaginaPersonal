I3 - Empatizando, creamos soluciones únicas

I3 es un proyecto creado junto a Salesianos que consiste en crear dispositivos de ayuda para todo tipo de personas.

Funciones destacadas:

  • Trabajo en grupo
  • Solución de problemas
  • Apostar por un futuro mejor
Multiwatch - Dispositivo de gestión de tiempo multi-plataforma.

Multiwatch es una herramienta diseñada para ayudar a los usuarios a gestionar su tiempo de manera eficiente.

Funciones destacadas:

  • Gestión del tiempo
  • Optimización de recursos
  • Medidor de sonido
Ibit - Página web informática

Ibit es una página web enfocada a la informática.

Servicios disponibles:

  • Venta de todo tipo de dispositivos informáticos
  • Mantenimiento y reparación de dispositivos informáticos
  • Web hosting
